# Mission

## Objective
Build a generic agent harness that can plan, research, implement, validate, and record progress using any coding agent.

## In scope
- One-task-per-iteration loop.
- Vendor-neutral agent adapter.
- Optional bounded research step.
- Validation gate runner.
- Durable progress tracking.

## Out of scope
- Direct deploy to production.
- Unbounded autonomous operation.
- Provider-specific UI features.

## Definition of done
- Backlog tasks marked complete all pass acceptance criteria.
- Validation commands pass.
- Research artifacts exist for tasks that require research.
- Progress log explains what changed and what remains.
